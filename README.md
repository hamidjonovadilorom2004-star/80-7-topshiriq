# 80-7-topshiriq
# Education Verification Blockchain (7-chi topshiriq)

Bu loyiha ta'lim muassasalari uchun diplom va sertifikatlarni blokcheynda yaratish, saqlash va ularning haqiqiyligini tekshirish uchun mo'ljallangan.

## Funksiyalar:
*   **Certificate Issuance**: Talabalarga diplom yoki sertifikatlarni blokcheynda yaratish (Faqat vakolatli shaxslar uchun).
*   **Authenticity Verification**: Sertifikatning asl nusxa ekanligini blokcheyn orqali istalgan kishi tekshirishi mumkin.
*   **Decentralized Storage**: Talaba ma'lumotlari xavfsiz va markazsiz holda saqlanadi.
*   **Access Management**: Talaba o'z ma'lumotlarini (diplomlarini) ish beruvchi yoki boshqa universitetga ochishi (Grant Access) mumkin.
*   **Unified Registry**: Barcha ta'lim natijalarini yagona va o'zgarmas tizimda jamlash.

## Steklar:
- **Solidity**: Ta'lim va avtorizatsiya mantiqi uchun.
- **React/Vite**: Talaba va admin paneli uchun.
- **Plus Jakarta Sans font**: Akademik va zamonaviy UI uchun mo'ljallangan dizayn.

## Ishga tushirish:
1. `cd 7_chi_topshiriq`
2. `npm install`
3. `npm run dev`

Loyiha to'g'irlangan shartlar asosida GitHub'ga yuklandi! ✅
